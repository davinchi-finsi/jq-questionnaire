jqQuestionnaire

jqQuestionnaire is a jquery widget for create static questionnaires

Features

  • Easy creation with PUG mixins
  • HTML full customizable
  • Modular
  • Enable or disable change of an option after check one
  • Immediate feedback
  • Points to sum if success and points to substract if fails
  • Cut off mark
  • Free

Options

let options = {
    delayOnAutoNext: 500,//delay before auto go to next question
    pointsForSuccess: 1,//amount of points to increase when an answer is correct
    pointsForFail: 0,//ammoint of points to substract when an answer is incorrect
    cutOffMark: 50,//minimum % of points to success (over 100%)
    immediateFeedback: false,// if true, when the user selects an option, the question will be checked immediately
    showCorrection: true, //show corrections at the end of the questionnaire
    showResult: true, //show the result of the questionnaire at the end
    multichoice: false,//if true, multiple options could be selected
    dialog: {//options for the result dialog. see jquery ui dialog options
        draggable: false,
        autoOpen: true,
        resizable: false,
        modal: true
    }
}

Simple use

jqQuestionnaire is modularized in parts, each part has a propose and could be place where you want inside of jqQuestionnaire container. Each part has a specific mixin in pug, de esta manera es posible manipular cada elemento por separado, añadir un id, añadir clases, encapsularlo en otro elemento html, etc Include the mixins from your instalation (node_modules or the folder in which jqQuestionnaire is

page.pug

include jqQuestionnaire/_jqQuestionnaire
+jq-questionnaire
    +jq-questionnaire__header
        +jq-questionnaire__title({
            content:"My questionnaire"
        })
        +jq-questionnaire__description({
            content:"Description for my questionnaire"
        })
        +jq-questionnaire__actions
            +jq-questionnaire__action({
                type:"start",
                content:"Start questionnaire"
            })
    +jq-questionnaire__body
        +jq-questionnaire__questions
            +jq-questionnaire__question
                +jq-questionnaire__statement
                    span Some question
                +jq-questionnaire__options
                    +jq-questionnaire__option({
                        name:"question1",
                        content:"Incorrect",
                        isCorrect:false
                    })
                    +jq-questionnaire__option({
                        name:"question1",
                        content:"Correct",
                        isCorrect:true
                    })
        +jq-questionnaire__feedback({
            type:"success",
            content:"Success feedback
        })
        +jq-questionnaire__actions
            +jq-questionnaire__action({
                type:"end",
                content:"End questionnaire"
            })
//- The container
